# Cozy Coding Club

A retro pixel-art personal website built by a young coder learning web development through vibe coding with [Claude Code](https://claude.ai/code).

**Live site:** [cozycodingclub.com](https://cozycodingclub.com)

## What's Here

- **Main site** — pixel-themed homepage with projects, blog, and about section
- **Calculator** (`/calc/`) — a working pixel-themed calculator app
- **Blog** — short posts about coding adventures

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript — no frameworks, no build tools
- Hosted on GitHub Pages with a custom domain
- DNS via Cloudflare

## Running Locally

Just open `index.html` in a browser. No dependencies to install.
